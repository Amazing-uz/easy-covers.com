# EasyCo - Сайт сравнения товаров 🍪

Современный и яркий сайт-блог для сравнения товаров с честными отзывами, анализом плюсов и минусов.

## 🎨 Концепция дизайна

### Цветовая схема (3 основных цвета + нейтралы)
- **Electric Lavender** (#9D50BB) - Основной цвет
- **Mango Tango** (#FF8235) - Вторичный цвет  
- **Cyber Mint** (#00FFC3) - Акцент для "Плюсов"
- **Hot Coral** (#FF4B2B) - Акцент для "Минусов"

### Типография
- **Заголовки:** Space Grotesk (футуристичный, смелый)
- **Текст:** Plus Jakarta Sans (четкий, читаемый)

## 📱 Структура страниц

### 🏠 Главная (`/`)
- **Hero Section** - Слайдер с "Versus" сравнениями товаров
- **Featured Products** - 3 топовых продукта в виде красивых карточек
- **Blog Section** - 4 последних статьи/гайда
- **CTA Section** - Подписка на рассылку

### ⚔️ Страница сравнения (`/comparison`)
- Полный анализ двух товаров (Кожа vs Силикон)
- **Split-screen дизайн:**
  - Левая сторона (Cyber Mint) - Плюсы
  - Правая сторона (Hot Coral) - Минусы
- Детальное сравнение с метриками
- Выводы и рекомендации

### 📚 Гайд (`/guide`)
- Полное руководство по выбору чехла
- Таблица сравнения материалов
- Рейтинги и шкалы оценок
- Советы по выбору

## ⚡ Ключевые компоненты

### ProConComparison
Разделенный экран с плюсами (зеленый) и минусами (красный):
```tsx
<ProConComparison 
  productName="Кожа vs Силикон"
  pros={[...]} 
  cons={[...]} 
/>
```

### ProductCard
Красивая карточка товара с градиентом, рейтингом, бейджем:
```tsx
<ProductCard 
  title="Кожаный чехол"
  emoji="🧥"
  color="from-amber-500"
  rating={5}
/>
```

### HeroSection
Интерактивный слайдер "Versus" для сравнения товаров

## 🎯 Дизайн подходы

1. **Hard Shadows** - Чёрные тени 4px смещения для объёма
2. **Brutalist-Light** - Четкие формы, максимум контраста
3. **Maximalist Colors** - Яркие, насыщенные цвета для энергии
4. **Split-screen** - Разделение плюсов/минусов на две стороны
5. **Package-like cards** - Карточки похожи на реальные товары

## 🚀 Как начать

1. Установка зависимостей (уже установлены):
```bash
pnpm install
```

2. Запуск dev сервера:
```bash
pnpm dev
```

3. Откройте http://localhost:3000

## 📝 Примеры контента

### Плюсы кожаного чехла
- Премиум ощущение в руках
- Улучшается с возрастом (патина)
- Отличная защита от падений
- Дышащий материал

### Минусы кожаного чехла
- Дорого ($50+)
- Требует регулярного ухода
- Может окрашивать пальцы
- Чувствителен к влаге

## 🎓 Навигация между страницами

- Главная → все секции интерактивны
- Hero слайдер → Читать сравнение → `/comparison`
- Продукты → клик по карточке → `/comparison` или `/guide`
- Blog посты → клик по посту → `/comparison` или `/guide`
