## 🍪 EasyCo - Сайт сравнения товаров

**Полностью готовый, модный сайт-блог для сравнения товаров с честными отзывами, анализом плюсов и минусов.**

---

## ✨ Что было создано:

### 🎨 Дизайн и визуал:
- ✅ Яркая цветовая схема (5 цветов: лавандовый, апельсиновый, мятный, коралловый + нейтралы)
- ✅ Две элегантные гарнитуры: Space Grotesk (заголовки) + Plus Jakarta Sans (текст)
- ✅ Brutal-light эстетика с hard shadows для объема
- ✅ Split-screen дизайн для плюсов (зеленый) и минусов (красный)
- ✅ Интерактивные карточки товаров с рейтингами и эмодзи

### 📱 Три полностью функциональные страницы:
1. **Главная (/)** - Слайдер сравнений, топ 3 товара, 4 блог-поста, подписка
2. **Сравнение (/comparison)** - Кожа vs Силикон с полным анализом плюсов/минусов
3. **Гайд (/guide)** - Полное руководство по выбору с таблицами и советами

### 🧩 8 кастомных компонентов:
- `Header` - Навигация с логотипом 🍪
- `HeroSection` - Интерактивный слайдер "Versus"
- `ProductCard` - Красивые карточки товаров
- `ProConComparison` - Split-screen плюсы/минусы
- `FeaturedProducts` - Сетка топ товаров
- `BlogSection` - Блог-посты
- `CTASection` - Подписка на рассылку
- `Footer` - Навигационный футер

### 🚀 Современный технологический стек:
- Next.js 16 с Turbopack (максимально быстро)
- React 19.2 с новыми хуками
- Tailwind CSS 4 (утилита-первый CSS)
- shadcn/ui компоненты
- lucide-react иконки

### 📊 Контент и примеры:
- ✅ Полное сравнение "Кожа vs Силикон" с реальными плюсами/минусами
- ✅ Примеры структуры блог-постов
- ✅ Система оценок (звезды, проценты, параметры)
- ✅ 3 сгенерированные изображения для визуала

### 📚 Полная документация:
- `README.md` - Основное руководство
- `DEPLOYMENT.md` - Инструкция по развертыванию на Vercel

## 🎯 Как начать:

### Локально:
```bash
pnpm install
pnpm dev
# Откройте http://localhost:3000
```

### На Vercel (рекомендуется):
1. Импортируйте проект на GitHub
2. На vercel.com нажмите "Import Project"
3. Выберите GitHub репозиторий
4. Нажмите Deploy
5. Готово! Ссылка появится через 1-2 минуты

## 💡 Советы по использованию:

1. **Изменить цвета:** Отредактируйте `/app/globals.css` переменные CSS
2. **Добавить товары:** Измените массивы в компонентах (hero-section, featured-products и т.д.)
3. **Новая страница:** Создайте файл в `/app/yourpage/page.tsx`
4. **Кастомный компонент:** Создайте файл в `/components/yourcomponent.tsx`
