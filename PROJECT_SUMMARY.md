## 🎯 Примеры использования:

### Добавить новый товар:
1. Отредактировать массив в `components/featured-products.tsx`
2. Добавить объект с title, emoji, color, rating

### Добавить блог пост:
1. Отредактировать массив в `components/blog-section.tsx`
2. Добавить заголовок, категорию, описание

### Изменить цвета:
1. Отредактировать `/app/globals.css`
2. Изменить CSS переменные (--electric-lavender и т.д.)

### Создать новую страницу:
1. Создать файл `/app/newpage/page.tsx`
2. Импортировать компоненты (Header, Footer и т.д.)
3. Построить layout
