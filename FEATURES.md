## 🎓 Кастомизация

### Легко изменяемые параметры
- ✅ Цвета в `globals.css`
- ✅ Шрифты в `layout.tsx`
- ✅ Контент в компонентах
- ✅ Маршруты (routes)

### Примеры кастомизации
```
1. Добавить товар - отредактировать компонент
2. Изменить цвета - обновить CSS переменные
3. Новая страница - создать файл в /app
4. Новый компонент - создать файл в /components
```
